Project 1 Report

  1. The website, while being based on a hobby of mine, is designed for an audience other than myself, that audience being less experienced players.
  2. The textual content was written by me from memory. I did use third-party images, crediting whenever possible.
  3. The site contains five pages, excluding this report.
  4. The site includes a header.
  5. The site has a navbar.
  6. I utilized media queries to make the navbar vertical rather than horizontal on smaller screens, and also divide the non-index pages into three columns on large screens.
  7. Styles are included in a shared stylesheet (styles.css), and the gallery CSS is in a separate stylesheet to keep things organized.
  8. Images are included in the form of the Logo on the top of each page, as well as a slideshow gallery for some of the individual pages.
  9. The favicon is declared, however refuses to show up on Chrome for some reason, but works on Edge.
  10. The stylesheets and script are organized into the css and js folders respectively.
  11. I did not use Bootstrap or similar frameworks.
  12. The pages passed the Nu Validator.
  13. The content is generally left-aligned, however the header is center-aligned to distinguish it.
  14. I have included padding in most cases.
  15. I used the Fonts Orbitron (for headings) and Raleway (for text), which I found on Google fonts.
  16. On the index page, I have included a contact form.
  17. On gtnh.html and impact.html, I have included links to the official websites for those modpacks.
  18. All of my class names use kebab-case and have meaningful names.
  19. I made a slideshow gallery using CSS and JavaScript to use on the info pages.
  20. I have shared the site in a repository accessible here.
  21. My file names are all lowercase.
  22. I have set up the Apache configuration to redirect http to https.
  23. I followed the tutorial "Forcing HTTPS" on twodee.org to do this.
  24. Apache has been configured to redirect.
  25. While it did seem like the commits were condensed, that was because I got a late start.
